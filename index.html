How to install MetaMask

  1. Go to metamask.io/download
  2. Choose your browser (Chrome, Firefox, Brave, Edge) or mobile
  3. Click "Install" and add the extension
  4. Create a new wallet or import an existing one
  5. Networks → enable "Show hidden networks" → select Sepolia
  6. Add Sepolia: Add Sepolia network
  7. Get test ETH: Google Cloud Faucet
